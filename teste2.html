Relatório de Testes Scrum - Casas Bahia

Teste Descrição
Teste de Usabilidade Usuários foram solicitados a navegar pelo site e fornecer feedback sobre a experiência de uso.
Teste de Performance O site foi submetido a diferentes cargas de tráfego para avaliar seu desempenho e tempo de resposta.
Teste de Compatibilidade Foram testados diferentes navegadores e dispositivos para garantir que o site funcione corretamente em todos.
Teste de Funcionalidade Cada funcionalidade do site foi testada individualmente para garantir que esteja funcionando conforme o esperado.
Teste de Segurança Foram identificadas e corrigidas vulnerabilidades de segurança no site para proteger os dados dos usuários.
Teste de Navegação Usuários foram instruídos a encontrar produtos específicos e concluir transações para avaliar a navegação do site.
Teste de Localização Foram avaliados os recursos de localização do site para garantir que sejam precisos e relevantes para os usuários.
Teste de Resiliência O site foi submetido a testes de estresse para avaliar sua capacidade de lidar com picos de tráfego e falhas de servidor.
